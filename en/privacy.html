Privacy Policy

Your privacy is our top commitment

Last updated: April 2026
🛡 Our Commitment

Workspace Container does not collect, store, or transmit any user data. The app runs entirely on your local machine, requires no internet connection, and contains no analytics or tracking code.

Data Collection

Workspace Container does not collect any form of user data, including but not limited to:

  • Personal information (name, email, phone number, etc.)
  • Device information (device model, OS version, etc.)
  • Usage data (feature usage frequency, operation logs, etc.)
  • Location information
  • Window content or screenshots

Network Access

Workspace Container does not make any network requests. The app functions fully without an internet connection. There are no server-side components, no cloud sync, and no remote configuration.

Accessibility Permission

Workspace Container requires macOS Accessibility permission to control the position and size of other application windows. The use of this permission is strictly limited to:

  • Reading window position and size information
  • Setting window position and size
  • Performing window minimize and restore operations

Workspace Container does not use Accessibility permission to read any window content, text input, or other sensitive information.

Third-Party Services

Workspace Container does not include any third-party SDKs, analytics tools, advertising frameworks, or tracking code.

Item Status
Third-party SDKs None
Analytics / Tracking None
Advertising None
Network Requests None
Data Collection None

App Store Privacy Label

In accordance with Apple's App Store privacy label requirements, Workspace Container's privacy label is "Data Not Collected."

Children's Privacy

Workspace Container does not collect any user data, and therefore does not collect personal information from children. The app is suitable for users of all ages (age rating 4+).

Policy Changes

If we make any changes to this privacy policy, we will post the updated version on this page with the revision date. Since Workspace Container does not collect user data, we cannot directly notify users of policy changes. We recommend checking this page periodically.
