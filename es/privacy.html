Política de privacidad

Tu privacidad es nuestro compromiso principal

Última actualización: abril de 2026
🛡 Nuestro compromiso

Workspace Container no recopila, almacena ni transmite ningún dato de usuario. La aplicación se ejecuta completamente en tu máquina local, no requiere conexión a Internet y no contiene código de análisis ni seguimiento.

Recopilación de datos

Workspace Container no recopila ningún tipo de dato de usuario, incluyendo pero no limitado a:

  • Información personal (nombre, correo electrónico, número de teléfono, etc.)
  • Información del dispositivo (modelo del dispositivo, versión del SO, etc.)
  • Datos de uso (frecuencia de uso de funciones, registros de operaciones, etc.)
  • Información de ubicación
  • Contenido de ventanas o capturas de pantalla

Acceso a la red

Workspace Container no realiza ninguna solicitud de red. La aplicación funciona completamente sin conexión a Internet. No hay componentes del lado del servidor, ni sincronización en la nube, ni configuración remota.

Permiso de accesibilidad

Workspace Container requiere el permiso de accesibilidad de macOS para controlar la posición y el tamaño de las ventanas de otras aplicaciones. El uso de este permiso se limita estrictamente a:

  • Leer la información de posición y tamaño de las ventanas
  • Establecer la posición y el tamaño de las ventanas
  • Realizar operaciones de minimizar y restaurar ventanas

Workspace Container no utiliza el permiso de accesibilidad para leer ningún contenido de ventanas, entrada de texto u otra información sensible.

Servicios de terceros

Workspace Container no incluye ningún SDK de terceros, herramientas de análisis, marcos publicitarios ni código de seguimiento.

Elemento Estado
SDKs de terceros Ninguno
Análisis / Seguimiento Ninguno
Publicidad Ninguno
Solicitudes de red Ninguno
Recopilación de datos Ninguno

Etiqueta de privacidad del App Store

De acuerdo con los requisitos de etiqueta de privacidad del App Store de Apple, la etiqueta de privacidad de Workspace Container es "No se recopilan datos".

Privacidad infantil

Workspace Container no recopila ningún dato de usuario y, por lo tanto, no recopila información personal de menores. La aplicación es apta para usuarios de todas las edades (clasificación por edad 4+).

Cambios en la política

Si realizamos algún cambio en esta política de privacidad, publicaremos la versión actualizada en esta página con la fecha de revisión. Dado que Workspace Container no recopila datos de usuario, no podemos notificar directamente a los usuarios sobre los cambios en la política. Recomendamos consultar esta página periódicamente.
