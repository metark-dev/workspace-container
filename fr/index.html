Trois étapes pour commencer

Si simple qu'aucun tutoriel n'est nécessaire

1
Lancez Workspace Container
Ouvrez l'application et la fenêtre du conteneur apparaît automatiquement. Accordez l'autorisation d'accessibilité au premier lancement.
2
Liez les fenêtres
Glissez n'importe quelle fenêtre d'application dans le conteneur, ou utilisez les raccourcis ⌃⌥1-4 pour lier rapidement.
3
Mettez-vous au travail
Déplacez le conteneur, toutes les fenêtres suivent. Redimensionnez le conteneur, les dispositions s'ajustent automatiquement. Concentrez-vous sur votre travail — laissez Workspace Container gérer les fenêtres.
