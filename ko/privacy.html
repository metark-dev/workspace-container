개인정보 보호정책

개인정보 보호는 저희의 최우선 약속입니다

최종 업데이트: 2026년 4월
🛡 저희의 약속

Workspace Container는 사용자 데이터를 수집, 저장 또는 전송하지 않습니다. 앱은 전적으로 로컬 기기에서 실행되며, 인터넷 연결이 필요 없고, 분석 또는 추적 코드를 포함하지 않습니다.

데이터 수집

Workspace Container는 다음을 포함하여 어떠한 형태의 사용자 데이터도 수집하지 않습니다:

  • 개인정보 (이름, 이메일, 전화번호 등)
  • 기기 정보 (기기 모델, OS 버전 등)
  • 사용 데이터 (기능 사용 빈도, 작업 로그 등)
  • 위치 정보
  • 윈도우 내용 또는 스크린샷

네트워크 접근

Workspace Container는 어떠한 네트워크 요청도 하지 않습니다. 앱은 인터넷 연결 없이 완전히 작동합니다. 서버 측 구성 요소, 클라우드 동기화, 원격 구성이 없습니다.

접근성 권한

Workspace Container는 다른 앱 윈도우의 위치와 크기를 제어하기 위해 macOS 접근성 권한이 필요합니다. 이 권한의 사용은 다음으로 엄격히 제한됩니다:

  • 윈도우 위치 및 크기 정보 읽기
  • 윈도우 위치 및 크기 설정
  • 윈도우 최소화 및 복원 작업 수행

Workspace Container는 접근성 권한을 사용하여 윈도우 내용, 텍스트 입력 또는 기타 민감한 정보를 읽지 않습니다.

서드파티 서비스

Workspace Container는 서드파티 SDK, 분석 도구, 광고 프레임워크 또는 추적 코드를 포함하지 않습니다.

항목 상태
서드파티 SDK 없음
분석 / 추적 없음
광고 없음
네트워크 요청 없음
데이터 수집 없음

App Store 개인정보 보호 라벨

Apple App Store 개인정보 보호 라벨 요구사항에 따라, Workspace Container의 개인정보 보호 라벨은 "데이터 수집 안 함"입니다.

아동 개인정보 보호

Workspace Container는 사용자 데이터를 수집하지 않으므로 아동의 개인정보도 수집하지 않습니다. 이 앱은 모든 연령대의 사용자에게 적합합니다 (연령 등급 4+).

정책 변경

이 개인정보 보호정책에 변경 사항이 있을 경우, 수정 날짜와 함께 이 페이지에 업데이트된 버전을 게시합니다. Workspace Container는 사용자 데이터를 수집하지 않으므로 정책 변경을 직접 알릴 수 없습니다. 이 페이지를 주기적으로 확인하시기 바랍니다.
