Política de Privacidade

Sua privacidade é nosso principal compromisso

Última atualização: Abril de 2026
🛡 Nosso Compromisso

O Workspace Container não coleta, armazena ou transmite nenhum dado do usuário. O aplicativo funciona inteiramente na sua máquina local, não requer conexão com a internet e não contém nenhum código de análise ou rastreamento.

Coleta de Dados

O Workspace Container não coleta nenhuma forma de dados do usuário, incluindo, mas não se limitando a:

  • Informações pessoais (nome, e-mail, número de telefone, etc.)
  • Informações do dispositivo (modelo do dispositivo, versão do SO, etc.)
  • Dados de uso (frequência de uso de recursos, logs de operação, etc.)
  • Informações de localização
  • Conteúdo de janelas ou capturas de tela

Acesso à Rede

O Workspace Container não faz nenhuma requisição de rede. O aplicativo funciona completamente sem conexão com a internet. Não há componentes do lado do servidor, sincronização na nuvem ou configuração remota.

Permissão de Acessibilidade

O Workspace Container requer a permissão de Acessibilidade do macOS para controlar a posição e o tamanho das janelas de outros aplicativos. O uso desta permissão é estritamente limitado a:

  • Leitura de informações de posição e tamanho das janelas
  • Definição de posição e tamanho das janelas
  • Execução de operações de minimizar e restaurar janelas

O Workspace Container não usa a permissão de Acessibilidade para ler qualquer conteúdo de janela, entrada de texto ou outras informações sensíveis.

Serviços de Terceiros

O Workspace Container não inclui nenhum SDK de terceiros, ferramentas de análise, frameworks de publicidade ou código de rastreamento.

Item Status
SDKs de terceiros Nenhum
Análise / Rastreamento Nenhum
Publicidade Nenhum
Requisições de Rede Nenhum
Coleta de Dados Nenhum

Rótulo de Privacidade da App Store

De acordo com os requisitos de rótulo de privacidade da App Store da Apple, o rótulo de privacidade do Workspace Container é "Dados Não Coletados".

Privacidade Infantil

O Workspace Container não coleta nenhum dado do usuário e, portanto, não coleta informações pessoais de crianças. O aplicativo é adequado para usuários de todas as idades (classificação etária 4+).

Alterações na Política

Se fizermos qualquer alteração nesta política de privacidade, publicaremos a versão atualizada nesta página com a data de revisão. Como o Workspace Container não coleta dados do usuário, não podemos notificar diretamente os usuários sobre alterações na política. Recomendamos verificar esta página periodicamente.
